# SafeWork MZ

Sistema de Gestão Interna de Ocorrências de SST

## ⚠️ Aviso Legal
Este software é exclusivo para registro e gestão interna de ocorrências de trabalho. 
NÃO emite, não substitui e não protocola a CAT oficial do INSS/MITESS.

A Comunicação de Acidente de Trabalho deve ser emitida pelo RH no portal oficial 
do INSS em até 24h após o acidente, conforme legislação de Moçambique.

## Funcionalidades
- Registro interno de ocorrências e quase-acidentes
- Relatórios gerenciais em PDF para departamento de SST
- Histórico e rastreabilidade de registros por empresa
- Dashboard de indicadores de segurança

SafeWork MZ não substitui obrigações legais junto ao MITESS/INSS.
